Description

The element_index contains a array to store the indices. Upon drawing using a element_index the array is transfered to the GPU.

element_index public construct/copy/destruct

  1. element_index();
    Construct an empty array. 
  2. explicit element_index(size_type size);
    Construct an array of a specific size. 
  3. ~element_index();
