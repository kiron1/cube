Dependencies

  • OpenGL
  • GLEW
  • FreeGLUT
  • Boost
  • C++11 compiler (e.g. clang-3.5)
  • CMake 3.0 or newer
  • optional BoostBook toolchaing to generate documentation

Configure

Configure options

Issue the following command in the source directoy to generate the project in the _build subdirectory (the _build directory will be automaticaly created it it does not exist). 

cmake -H. -B_build
Note

The Ninja build system is recommended, e.g. add -GNinja to the above command.

Configure options

The following configure options are available:

  • CMAKE_BUILD_TYPE - Change build type.
  • CMAKE_INSTALL_PREFIX - Change installation location.
  • BUILD_TESTING - Enable generation of test targets.
  • BUILD_DOCUMENTATION - Enable generation of doc target.
  • Cube_DEVELOPER - Enable custom targets for developers (e.g. clang-format).

To change some options, append -DOPTION=Value (e.g. -DCMAKE_BUILD_TYPE=Debug to generate the project to build in debug mode).

Compile

The following command will build the project: 

cmake --build _build --config Release

Install

To install the project, use the following command: 

cmake --build _build --config Release --target install
