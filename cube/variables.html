Variables

The OpenGL variables proxie classes allow access to the internal variables of a OpenGL cube::gloo::program.

Usage

It is recomended to use one of the available typedef:

  • attribute::float_
  • attribute::vec2
  • attribute::vec3
  • attribute::vec4
  • uniform::float_
  • uniform::vec2
  • uniform::vec3
  • uniform::vec4
  • uniform::mat2
  • uniform::mat3
  • uniform::mat4
